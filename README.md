# Team Sweet Workshop

## Про проєкт

**Team Sweet Workshop** — командний вебпроєкт, створений відповідно до
дизайн-макету та технічного завдання.

Сайт має адаптивну верстку для:

- мобільних пристроїв;
- планшетів;
- десктопів.

---

## Основні можливості

- Адаптивна семантична верстка
- Модальна форма зворотного зв’язку
- Відправка форми
- Слайдери та інтерактивні елементи
- Валідація полів форми
- Робота з API та HTTP-запитами

---

## Запуск проєкту

### Встановлення залежностей

```bash
npm install
```

### Запуск локального сервера

```bash
npm run dev
```

### Збірка проєкту

```bash
npm run build
```

### Перегляд production-збірки

```bash
npm run preview
```
